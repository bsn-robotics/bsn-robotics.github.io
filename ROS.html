Стек навигации
Для использования узла move_base  навигационного стека, нам нужно иметь глобальный планировщик и локальный планировщик.

Существуют три глобальных планировщика, которые придерживаются интерфейса nav core :: BaseGlobal Planner: carrot planner, navfn и global planner.

 
http://zkytony.com/documents/navguide.pdf
To use the move base node in navigation stack, we need to have a global planner and a local planner. There are three global planners that adhere to nav core::BaseGlobal Planner interface: carrot planner, navfn and global planner.
