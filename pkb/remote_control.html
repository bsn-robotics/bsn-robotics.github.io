Удаленное управление Windows -> Ubuntu

Компьютеры подключаются к точке доступа Wi-fi.

На Ubuntu запускается SSH сервер:

https://gamelton.com/2011/08/08/vnc-via-ssh/

https://losst.ru/ustanovka-ssh-ubuntu-16-04

Устновка ssh-сервера на ubuntu 16.04

sudo apt install openssh-server

service ssh status
ssh start/running, process 2006

Проверка открытости 22 порта (порт, используемый по умолчанию SSH):

netstat -tulpan | grep :22


Автозагрузка для 16.04:

sudo systemctl enable sshd

sudo systemctl disable sshd

На Windows запускается SSH клиент, Putty.

x11vnc

Итак, удаляем vino, в Ubuntu и Debian это:

sudo apt-get remove vino

Устанавливаем x11vnc

sudo apt-get install x11vnc

Генерируем пароль для доступа:

x11vnc -storepasswd

После этого пароль по-умолчанию сохраняется в /home/USERNAME/.vnc/passwd (где USERNAME – имя пользователя, под которым вы выполнили команду).

Теперь нам надо добавить автозагрузку сервера.
