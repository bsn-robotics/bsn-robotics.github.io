Управление с помощью клавиатуры

Нужно подключить плагин в SDF файле, и установить пакет в ROS.

sudo apt-get install ros-kinetic-teleop-twist-keyboard
rosstack profile
rospack profile

rosrun teleop_twist_keyboard teleop_twist_keyboard.py

Вид с виртуальной камеры

Нужно подключить плагин в SDF файле, и установить пакет в ROS.

rosrun image_view image_view image:=/robot/camera1/image_raw


Моделирование turtlebot2 вместе с rtabmap.

http://wiki.ros.org/rtabmap_ros/Tutorials/MappingAndNavigationOnTurtlebot

http://learn.turtlebot.com/2015/02/03/8/

Автономная навигация с rtabmap

Команды для запуска симуляции:

roslaunch turtlebot_gazebo turtlebot_world.launch
roslaunch rtabmap_ros demo_turtlebot_mapping.launch simulation:=true
roslaunch rtabmap_ros demo_turtlebot_rviz.launch

Автономная навигация без rtabmap

Для того чтобы запустить автономную навигацию c amcl

roslaunch turtlebot_gazebo turtlebot_world.launch

roslaunch turtlebot_gazebo amcl_demo.launch

roslaunch turtlebot_rviz_launchers view_navigation.launch

Gmapping

roslaunch turtlebot_gazebo turtlebot_world.launch

roslaunch turtlebot_gazebo gmapping_demo.launch

roslaunch turtlebot_rviz_launchers view_navigation.launch

move_base при этом не запускается.
