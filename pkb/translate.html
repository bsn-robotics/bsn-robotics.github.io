Выполнение скрипта по нажатию сочитания клавиш


Вариант №1

Получаем ключ API для запросов перевода от Яндекс https://translate.yandex.ru/developers/keys

Скрипт:

#!/bin/sh
key="вставляем сюда свой ключ API"
text="$(echo $(xsel -o) | sed "s/[\"\'<>]//g")"
translate="$(wget -qO - "https://translate.yandex.net/api/v1.5/tr.json/translate?key=$key&text=$text&lang=ru")"
notify-send -u critical "$text" "$(echo $translate | sed 's/.*\[\"\(.*\)\"\].*/\1/')"

Называем, например translate и помещаем в /usr/bin/, задав свойство на исполнение:

sudo chmod 777 ./translate && sudo mv ./translate /usr/bin/

Вариант №2

С использованием https://github.com/soimort/translate-shell

Установим зависимости:

sudo apt-get install gawk

Скрипт:

#!/bin/sh
notify-send -u critical "$(xsel -o | trans -no-ansi -l ru)"

Называем, например translate и помещаем в /usr/bin/, задав свойство на исполнение:

sudo chmod 777 ./translate && sudo mv ./translate /usr/bin/
