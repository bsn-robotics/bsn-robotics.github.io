Cоздал проект для СLion SFML 2.2, 2.4 не линкуется на Ubuntu 14.04, потому что требуется gcc версии > 4.8 .

Устанавливается 2.2 через sudo apt-get install libsfml-dev.

2.4 качается с сайта и рспихивается по папкам в /usr/local/include, /usr/local/lib, /usr/local/share.
